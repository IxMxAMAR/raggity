# Security Policy

## Supported Versions

Only the latest minor release of raggity receives security fixes.

| Version | Supported |
|---------|-----------|
| latest  | Yes       |
| older   | No        |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report security issues by email to **officialamrendrasingh@gmail.com** with the subject line `[raggity] Security Report`.

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- raggity version (`rag --version`), Python version, and OS

You will receive an acknowledgement within 48 hours. Once the issue is confirmed and a fix is available, a patched release will be published and the issue may be publicly disclosed.

Thank you for helping keep raggity secure.
