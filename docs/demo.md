# Demo

The GIF below shows raggity in action: ingesting a knowledge base, answering a question with verified source citations, and demonstrating selective abstention when nothing in the index is relevant.

![raggity demo](https://raw.githubusercontent.com/IxMxAMAR/raggity/master/docs/demo.gif)

> **Note:** The GIF is generated by the maintainer — it is not checked in as a binary. If `docs/demo.gif` is missing, regenerate it locally (see below).

---

## Regenerate the demo GIF

Requires [VHS](https://github.com/charmbracelet/vhs) (`brew install vhs` / `go install github.com/charmbracelet/vhs@latest`).

```bash
# From the repo root:
vhs demo.tape
# → writes docs/demo.gif
```

The tape script is at [`demo.tape`](https://github.com/IxMxAMAR/raggity/blob/master/demo.tape) in the repository root. Edit it to adjust the commands, timing, or terminal theme.
